Modem Device

 ->  Network Device   
 -> Stand for Modulator & De-Modulator
 -> Device that is placed between computer systems
         & telephones lines. 

 -> It has two Parts 
      ->Modulator (convert digital signal to analog wave) 
      ->De-Modulator (convert analog to digital signals)
-> Modem directly connects to ISP

OSI Model

Seven layers

  1. Physical Layer
  2. Data Link Layer
  3. Network Layer
  4. Transport Layer
  5. Session Layer
  6. Presentation Layer
  7. Application Layer

TCP/IP Model


--> The TCP/IP model is a conceptual model used to describe the communication protocols used on the internet
--> It consists of four layers:
      --> the application layer,
      --> the transport layer, 
      -->  the internet layer, 
      -->  and the link layer. 

---> Each layer performs specific functions in the process of transmitting data over a network.
